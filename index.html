HTML 태그

태그 설명 예시
<!-- --> 주석을 정의 <!--이 구간은 실행이 안됩니다.-->
<!DOCTYPE> 해당 문서의 타입을 정의 <!DOCTYPE HTML>
<body> 해당 문서의 콘텐츠 영역을 정의 <body>HTML문서의 콘텐츠 영역입니다.</body>
<br> 텍스트 내의 줄바꿈을 정의, 종료 태그가 없다. 안녕<br>반가워
<button> 클릭할 수 있는 버튼을 정의 <button type="submit">등록완료</button>
<div> HTML문서에서 특정 영역이나 구획을 정의 <div>{background-color: orange;}</div>
<h1>~<h6> HTML문서에서 제목을 정의, <h1>요소가 가장 중요한 제목 <h1>이 텍스트는 h1요소를 사용한 제목입니다.</h1>
<html> HTML문서의 루트 요소를 정의 <html lang ="ko">
<input> 사용자로부터 입력을 받을 수 있는 입력 필드를 정의 <input type = "text" name="st_name">
<label> 사용자 인터페이스 요소의 라벨을 정의.
for속성을 사용해 다른 요소와 결합할 수 있으며, for속성값은 결합하고자 하는 요소의 id속성값과 같아야한다.		 <input type="radio" name="ages" id="twenty" value="twenties">
<label for="twenty">20대</label>
<title> 해당 문서의 제목을 정의 <title>HTML문서의 제목입니다.</title>
